# 雨轩地图标点（独立插件）

这个目录是一个可独立运行的插件。

## 功能

- 自定义点位添加、删除、显示
- 地图坐标快速输入并设置导航
- 点位导入/导出
- 账号级共享点位（角色互通）
- 简中/英文国际化

## 安装

将整个 `YuXuanMapPoint` 目录放到：

`World of Warcraft/_retail_/Interface/AddOns/`

确保目录内有 `YuXuanMapPoint.toc`。

## 命令

- `/yxpin`：快速添加当前位置点位
- `/雨轩标点`：快速添加当前位置点位
- `/yxmp`：打开设置

## 保存变量

- `YuXuanMapPointDB`（账号级）
